HTML CSS and Javascript

In a webpage HTML provides the structure and CSS provides the presentation. HTML tells you where everything goes and what item belongs to which part while css tells each item how it looks. Colour, size and position are just a few of the ways css can affect the presentation. Like a house, html would be the structure while css is the paint.

A program is a set of instructions that tells the computer what to do and control flow defines when each instruction is carried out. Usually the instructions are carried out sequentially like going from instructions 1 to 10, but there are sections where conditions are checked which can jump around the instruction set depending on the outcome. An example of this is the for loop, for example instructions 1 to 5 deals with washing a bunch of dishes.

  1. Pick up a dirty dish.
  2. Scrub the dish.
  3. Rinse the dish.
  4. Put away the dish.
  5. Check if there are any dirty dishes left, if there are goto 1 else goto 6.
  6. Relax.

In this case, the instructions will never move past 5 until all the dishes are clean and put away.

The Document Object Model or DOM is a representation of a webpage, it takes the content and structures it in such a way that programs are able to interact with it. One way to use the DOM is adding and removing content from a page such as finding all items of a certain type and putting a border around them. All of this and more can be accomplish through javascript.

To access data in an array you provide it the index such as asking for the data in position 3 of the array. Do keep in mind that array indexes start counting from 0 so to get the data on position 3 the index is 2 array[2] and on position 1 the index is 0 array[0]. Data in objects is associated with keys, and to access the data a key is provided to the object in either dot object.key or bracket object[key] notations.

Functions are a set of related instructions grouped together allowing them to be reused. Going back to the dishwashing example earlier, rather than writing the same instructions everytime you want to wash some dishes you can instead group those instructions together in a function called washDishes and call it anytime dishes need washing.
